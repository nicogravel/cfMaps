# **Population Receptive Field (pRF) mapping**


This pipeline relies on `CUDA` (with `tensorflow`and `cuda` v.12.8 in an NVIDIA RTX 1000 Ada GPU). However, in the abscenece of a correctly installed nvidia card, `tensorflow` defaults to CPU parallelization. For the initial grid search, `prfpy` also uses CPU parallelization and is threrefore quite fast.
